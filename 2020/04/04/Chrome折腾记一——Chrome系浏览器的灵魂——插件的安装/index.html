说到上网，每个人的电脑上都一定会有“浏览器”这一类软件。

国内比较常见的浏览器有QQ、搜狗、360、百分等等，这些都是chromium内核的浏览器即chrome系，换句话来说，就是谷歌浏览器换了个皮肤，还慢的一批，不过再怎么说他们本质上还是谷歌浏览器，还是可以安装谷歌浏览器支持的各种插件的。

什么？你和我说你不知道插件是什么？
你这么多年网白上了。

诶，等等，别走别走，开个玩笑嘛，这就教你(o゜▽゜)o☆

—————————————正文开始———————————————
别的咱也不多说了，直接从插件（扩展）的安装开始。

各种浏览器的插件安装方法都是都是大同小异的，基本流程都是：商店 → 搜索 → 安装 → 结束。

前提是你是自带梯子的巴啦啦小魔仙可以访问chrome商店。

这时就小伙伴会问了：“诶Paxton你说的不对啊，我的XX浏览器都有商店啊？”

但是这些第三方插件商店里面的插件普遍都是又少又旧，有些甚至常用插件都搜不到。

因此在这里我给大家推荐一个网站：https://www.gugeapps.net/webstore/new


推荐网站

官方商店（下图）

是不是有内味了( •̀ ω •́ )✧简直亲兄弟好吧，虽然官方安装不用那么麻烦

首先我们先来搜索一个谷歌浏览器必备的插件adb block

直接按照网站的指引进行下载即可，亲测无毒无捆绑

接下来以QQ浏览器进行举例

下载完成后，我们有两种办法进行安装。

方法一：拖拽安装法

首先将我们得到的这个压缩文件的后缀名.zip 更换为 .crx（没有显示后缀名的自行百度解决）

然后找到QQ浏览器的右上角的三杠

选择应用中心进入，在页面上方选择管理我的应用

接着返回桌面拖动你下载的crx文件到这个页面松开鼠标，就可以成功安装了

提示：此方法仅适用于部分浏览器以及部分插件

方法二：解压安装法

还是看回我们刚刚下载的.zip压缩包

右键选择解压到……就可以得到下图所示的一个文件夹


注意这个文件夹一定要是打开直接可以看到上图所示的文件的，不能是像下图这样的套娃文件夹
]
如果是这种套娃文件夹就需要手动把里面的文件拖出来

接着我们打开浏览器，这里以edge chrome版浏览器进行举例

还是找到你的浏览器的扩展管理页

注意一定要把开发者模式打开，然后选择加载解压缩的扩展

选择后就可以在浏览器的扩展管理页面看到他啦

这种方法适用于所有chrome系浏览器

还有一点要注意，就是解压出来的那个文件夹是绝对不能删掉的，删掉了扩展就没了，因此在安装浏览器扩展的时候一定要选好一个位置集中放置扩展的文件。

其实还有传说中的最简便的第三种方法，我们下期Chrome折腾记再聊(。・ω・。)
